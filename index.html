What Are Thinkies?

Thinkies are pattern-matching tools that help you generate creative ideas. Each Thinkie contains a pattern (something you recognize) and a transformation (something you do with that pattern). They're not about producing good ideas directly—they're about producing lots of ideas, some of which might be brilliant.

Pattern Recognition

Thinkies teach you to spot common situations where creative thinking can break through barriers.

Simple Transformations

Each Thinkie provides a specific transformation to apply when you recognize the pattern.

Deferred Judgment

Thinkies separate idea generation from evaluation, a key principle of creative thinking.

How Thinkies Work

  1. Match the pattern in a situation
  2. Apply the transformation
  3. Evaluate the result afterward

Can't/Because

Pattern: "We can't X because Y."

Transform: "When (not Y), then we can X."

Example: "We can't deploy daily because there are too many bugs" → "When we have fewer bugs, we can deploy daily."

Easy Bit

Pattern: You're paralyzed by the magnitude of a problem.

Transform: Pick the absolutely easiest helpful thing to do and do it.

Example: Facing a complex project, start by setting up the simplest component.

Half in Half

Pattern: Someone gives you a date for a scope of work.

Transform: Ask which half they can see in half the time.

Example: "We'll deliver in 6 months" → "Which half can I see in 3 months?"
